- **three.js r180** — WebGL 3D library powering the two particle scenes (hero/download "main particles" and try-solutions "morphing particles"); bundled locally as an ES module under `assets/js/vendor/`.
- **WebGL GPGPU particle simulation** — ping-pong render targets + custom GLSL shaders (simplex noise, ring-displacement and image-morphing fields) ported verbatim from the original Angular bundle into `assets/js/particles.js`.
- **Poisson-disk sampling** — algorithm reimplemented in `particles.js` to generate the particle point cloud; also runs inside a Web Worker to derive image-shaped point sets for the morphing scene.
- **Web Workers** — off-main-thread computation of nearest-point data for the morphing particle textures.
- **Angular 20.3.11** — original SPA framework (server-rendered DOM was used as the layout source; the runtime itself is not shipped in this extraction).
- **Google Sans Flex** — variable display/body typeface (loaded via `@font-face`).
- **Google Sans Code** — monospaced typeface used for code-style accents.
- **Google Symbols** — icon font; all UI glyphs (chevrons, download, terminal, play, etc.) are text ligatures.
- **CSS custom properties** — design-token system (palette, theme surfaces, spacing scale, grid) defined in the original Angular global stylesheet `styles-7KLEMMT6.css`.
- **CSS Grid + flexbox** — responsive 12-column grid and section layouts.
- **IntersectionObserver** — pauses/resumes each WebGL scene when its section enters or leaves the viewport.
- **HTML5 video** — autoplaying muted background videos in the hero and the Antigravity IDE feature.
